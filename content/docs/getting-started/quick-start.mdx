## First Run

Run Syntra in your project directory:

```bash
npx syntra init
```

## Quick Steps

1. Run `npx syntra init` to initialize Syntra in your project.
2. Ask your AI agent to analyze the project and update the architecture instructions:
   ```
   /task Analyze this project carefully and update ./ai/instructions/architecture.instructions.md accordingly.
   ```
3. Run `npx syntra sync` to sync your configuration.
4. Start your first task with `/TASK-1`.
5. Your project is set up!
